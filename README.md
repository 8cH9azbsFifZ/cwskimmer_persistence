# cwskimmer_persistence

Connects to the telnet client of CW SKIMMER and writes to CW SPOTS to a mongodb.

# Setup
- Configure connection details in docker-compose.yml.
- Start a mongo db.
- Start a cw skimmer.

# Running
- `docker-compose up -d`
